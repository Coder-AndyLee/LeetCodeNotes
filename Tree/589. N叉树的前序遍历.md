# Description
给定一个 N 叉树，返回其节点值的前序遍历。

例如，给定一个 3叉树 :

返回其前序遍历: [1,3,5,6,2,4]。


# Solution
## 1. 递归
- 基于递归方法的先序遍历
```python
"""
# Definition for a Node.
class Node:
    def __init__(self, val=None, children=None):
        self.val = val
        self.children = children
"""

class Solution:
	def VLR(self, root):
		if root:
			self.ans.append(root.val)
			for i in root.children:
				VLR(i)
    
	def preorder(self, root: 'Node') -> List[int]:
        self.ans = []
        self.VLR(root)
        return self.ans
```
## 2. 循环
- 用栈（准确地说是个list）保存待遍历节点
- 需要注意存节点入栈的顺序
```python
"""
# Definition for a Node.
class Node:
    def __init__(self, val=None, children=None):
        self.val = val
        self.children = children
"""

class Solution:
    def preorder(self, root: 'Node') -> List[int]:
        if not root:
            return []
        else:
            ans = []
            stack = [root]
            while len(stack) > 0:
                current = stack.pop(0)
                if not current:
                    continue
                ans.append(current.val)
                stack = current.children + stack
            return ans
```
